📊 Pivot Tables

تحليل البيانات بطريقة احترافية في Excel

🧠 الشرح

Pivot Table هي أداة قوية في Excel كتحول البيانات الكبيرة إلى ملخصات سهلة الفهم (تقارير وإحصائيات).

📌 فين كنستعملو Pivot؟

  • 📊 تحليل المبيعات
  • 👨‍🎓 تحليل نتائج الطلاب
  • 💰 تقارير مالية
  • 📈 إحصائيات سريعة

🎥 فيديو شرح

🎯 شرح Pivot Tables في Excel

📊 مثال مبسط

نفترض عندك مبيعات:

Product | Sales
A       | 100
B       | 200
A       | 150
B       | 50

Pivot Table غادي يعطيك:

A = 250
B = 250
